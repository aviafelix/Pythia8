Onia Processes

Production of J/psi or Upsilon, directly and via chi states and the colour-octet mechanism. In each process the square-bracketed expression specifies the state in spectroscopic notation, (2S+1) L J, followed by (1) for colour-singlet states and (8) for colour-octet ditto.

The original Fortran code for these processes has been contributed by Stefan Wolf [unpublished]; for the C++ version only the unpolarized expressions are retained.

The description of final-state radiation is in this case based on some further model assumptions.

Onium matrix elements

The implementation of charmonium and bottomonium production, including the colour-octet production mechanism, requires information on NRQCD matrix elements for the various wavefunctions involved. Default values for these are encoded in the following ten variables. They are taken from [Nas00]; see also [Bar06].

parm name="SigmaQCD:OJpsi3S11" default="1.16" min="0.0"
<O(J/psi)[3S1(1)]>.

parm name="SigmaQCD:OJpsi3S18" default="0.0119" min="0.0"
<O(J/psi)[3S1(8)]>.

parm name="SigmaQCD:OJpsi1S08" default="0.01" min="0.0"
<O(J/psi)[1S0(8)]>.

parm name="SigmaQCD:OJpsi3P08" default="0.01" min="0.0"
<O(J/psi)[3P0(8)]>/m_c^2.

parm name="SigmaQCD:Ochic03P01" default="0.05" min="0.0"
<O(chi_c0)[3P0(8)]>/m_c^2.

parm name="SigmaQCD:OUpsilon3S11" default="9.28" min="0.0"
<O(Upsilon)[3S1(1)]>.

parm name="SigmaQCD:OUpsilon3S18" default="0.15" min="0.0"
<O(Upsilon)[3S1(8)]>.

parm name="SigmaQCD:OUpsilon1S08" default="0.02" min="0.0"
<O(Upsilon)[1S0(8)]>.

parm name="SigmaQCD:OUpsilon3P08" default="0.48" min="0.0"
<O(Upsilon)[3P0(8)]>/m_b^2.

parm name="SigmaQCD:Ochib03P01" default="0.09" min="0.0"
<O(chi_b0)[3P0(8)]>/m_b^2.
